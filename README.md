# Dawson Personal Knowledge Base

A structured repository for capturing knowledge, patterns, prompts, and learnings across AI, cloud, infrastructure, and beyond.

## Navigation

| Section | Description |
|---------|-------------|
| [ai/](./ai/) | AI tools, prompts, agents, and models |
| [cloud/](./cloud/) | Cloud platforms - Azure, AWS, GCP |
| [devops/](./devops/) | CI/CD, containers, Kubernetes, IaC |
| [til/](./til/) | Today I Learned - quick daily entries |
| [references/](./references/) | Cheatsheets, links, and quick references |
| [_templates/](./_templates/) | Templates for new entries |

## How to Use This PKB

- **Quick learnings** → `til/` with date prefix e.g. `2026-02-22-how-mcp-works.md`
- **Deep dives** → appropriate section folder with a descriptive README
- **Prompts** → `ai/prompts/` organized by use case
- **Patterns** → `ai/agents/patterns/` or `cloud/patterns/`
- **References** → `references/cheatsheets/` for quick lookups

## Principles

- Capture it when you learn it
- Link related concepts
- Include working examples where possible
- Note what didn't work as well as what did
